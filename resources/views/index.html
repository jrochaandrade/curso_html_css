Superscript and subscript (sobrescrito e subscrito)

Para usar em formulas ex. 22 ou quando quiser que venha abaixo ex. H2O, deve se colocar para o sobrescrito (acima) dentro da tag <sup>DIGITE AQUI</sup> e para o subscrito (abaixo) dentro das tags <sub>DIGITE AQUI</sub>

Bbold and italic (negrito e itálico)

Para adicionar uma palavra em negrito coloque ela dentro das tags <b> DIGITE AQUI </b>

E para colocar um palabra em itálico colocque ela dentro das tagas <i> DIGITE AQUI </i>

Porém essa não são as formas corredas de dar ênfase em uma palavra. Devemos utilizar marcação semântica.

Tags Strong (forte) e Emphasis (ênfase)

Usa-se a tag strong quando queremos demonstar que aquela palavra ou fraze tem um forte importância, vamos usar a tag strong no lugar do negrito, tendo em vista que a palavra vai ficar negrito também colocando tudo dentro das tags <strong>DIGITE AQUI</strong>, e emphasis para dar ênfase em uma palavra ou frase no lugar do itálico, tendo em vista que ela vai ficar em itálico colocando tudo dentro das tags <em>DIGITE AQUI</em>

Quotations (citações)

Para fazer citações longas use a tag blockquote coloque tudo dentro das tags <blockquote cite="AQUI VAI O LINK DA CITAÇÃO">DIGITE A CITAÇÃO AQUI</blockquote> o texto vai ficar como o exemplo a seguir;

Lorem ipsum dolor sit amet consectetur, adipisicing elit. Sed at vel nobis itaque, explicabo eaque maxime aperiam facere totam libero nisi voluptatum sit in ducimus ratione ea suscipit neque iusto.

Para citações curtas use a tag q colocando dentro das tags <q>DIGITE AQUI</q> conforme o próximo exemplo; Fulano de tal disse aqui vai um exemplo de citação curta.

Abreviations and acronims (abreviações e acrônimos)

Para usar uma abreviação usamos a tag abbr preechendo o titulo com o significa da abreviação ou acrônimo. ex:

O <abbr title="Professor">Prof</abbr> Gustavo Guanabara fez um cuso de HTML no canal...

O Prof Gustavo Guanabara fez um curso de HTML no canal Curso em vídeo.

Também se usa a tag abbr em acrônimos ex: Os EUA era a grande potência mundial!

Citations and definitions (citações e definições)

Para fazer refência a um trabalho como um livro, filme ou trabalho de pesquisa use a tag cite ex: A Casa do Dragão by George R.R. Martin. Colocando dentro da tag <cite>DIGITE AQUI</cite>

Para definir uma nova terminologia ou um novo jargão se usa dfn ex: Um buraco negro é uma região do espaço da qual nada, nem mesmo a luz, pode escapar. colocando a frase ou palavra dentro das tags <dfn>DIGITE AQUI</dfn>

Author details (detalhes do autor)

                    <p><a href="mailto:emailteste@gmail.com">emailteste@gmail.com</a></p>
                    <p>896 Rua das Pedras, Araça.</p>

Changes to content (mudança de conteúdo)

Usamos os elementos ins e del quando queremos demonstar um conteúdo que foi alterado no texto ex: Era a pior melhor idéia que ela tinha.

Para marcar algo que vamos retirar usamos a tag <del>DIGITE AQUI</del> e para adicionar usamos a tag <ins>DIGITE AQUI</ins>

Usmos também a tag s para indicar um conteúdo que não é mais relevante ou preciso mas que não deve ser deletado deve ser colocado dentro das tags <s>DIGITE AQUI</s> ex:

Computador:

Era R$9.999,00.

Agora é R$999,99

Ordered lists (listar ordenadas)

Para criar uma lista ordenadas colocamos dentro das tags <ol>DIGITE AQUI</ol> e dentro dessa tag cada item vai dentro de outra tag <li>CONTEÚDO DAS LISTAS</li>

Nesso código também usamos a tag pre que deixa o texto com a mesma identação do código.

        <pre>
            <ol>
                <li>Acordar</li>
                <li>Tomar café</li>
                <li>Estudar</li>
                <li>Trabalhar</li>
                <li>Dormir</li>
            </ol>
        </pre>

  1. Acordar
  2. Tomar café
  3. Estudar
  4. Trabalhar
  5. Dormir

Para mudar o estilo da lista basta acrescentar o type="Pode ser 1, A, a, I e i"

        <pre>
            <ol type="A">
                <li>Acordar</li>
                <li>Tomar café</li>
                <li>Estudar</li>
                <li>Trabalhar</li>
                <li>Dormir</li>
            </ol>
        </pre>

  1. Acordar
  2. Tomar café
  3. Estudar
  4. Trabalhar
  5. Dormir

E se quiser começar de uma numero ou letra expecifico basta acrescentar start="3" após o st 

        <pre>
            <ol type="1" start="3">
                <li>Acordar</li>
                <li>Tomar café</li>
                <li>Estudar</li>
                <li>Trabalhar</li>
                <li>Dormir</li>
            </ol>
        </pre>

  1. Acordar
  2. Tomar café
  3. Estudar
  4. Trabalhar
  5. Dormir

Definition lists (listas de definições)

Quando queremos usar um série de termos e suas definições usamos as tags dl e dentro dela colocamos a tag dt para conter o termo e dd para sua definição, ex:

        <dl>
            <dt>Programação</dt>
            <dd>Programação é o processo de criação de um conjunto de instruções que dizem ao computador como realizar uma tarefa. Programação pode ser
 feita usando uma variedade de "idiomas" do computador, tais como SQL, Java, Python e C++.</dd>
        </dl>
        <dl>

 sistemas operacionais, redes sociais, aplicativos de celular e outros dispositivos, entre outras infinitas possibilidades.</dd>
        </dl>

Programação
Programação é o processo de criação de um conjunto de instruções que dizem ao computador como realizar uma tarefa. Programação pode ser feita usando uma variedade de "idiomas" do computador, tais como SQL, Java, Python e C++.
Desenvolvedor
O desenvolvedor é o profissional que escreve e cria softwares, que podem ser websites, programas de computadores pessoais ou empresariais, sistemas operacionais, redes sociais, aplicativos de celular e outros dispositivos, entre outras infinitas possibilidades.

Tag Pré-Formatado

Para imprimir um texto na tela que você precisa que fique aninhado conforme o código, coloque tudo dentro da tag <pre> Digite aqui </pre>.


            <pre>
                num1 = float(input("Digite o primeiro número: "))
                num2 = float(input("Digite o segundo número: "))
        
                operacao = input("Deseja somar (+) ou subtrair (-) os números? ")
        
                if operacao == "+":
                    resultado = num1 + num2
                    print("O resultado da soma é:", resultado)
                elif operacao == "-":
                    resultado = num1 - num2
                    print("O resultado da subtração é:", resultado)
                else:
                    print("Operação inválida. Por favor, escolha '+' para somar ou '-' para subtrair.")
            </pre>

Para adicionar imagens na minha pagina vou usar a tag <img src="Aqui vai o caminho da imagens" alt= "Descrição da imagem">

Também é possivel adicionar uma imagem direto da internet, <img src="URL da imagem" alt= "Descrição da imagem"> no lugar do caminha coloque a URL da imagem.


Imagens dinâmicas

Para adicionar imagens que se alteram de acordo com o tamanho da tela faça conforme o código abaixo:

            <picture>
                <source media="(max-width: 500px)" srcset="/pictures/foto-p.png" type="image/png">
                <source media="(max-width: 1050px)" srcset="/pictures/foto-m.png" type="image/png">
                <img src="pictures/foto-g.png" alt="Imagem flexível">
            </picture>
